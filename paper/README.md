# CQT Manuscripts

This directory contains the official manuscripts of the **Coherent Quantum Field Theory (CQT)**.

## Available Versions

English version  
https://doi.org/10.5281/zenodo.19100362

German version  
https://doi.org/10.5281/zenodo.19100136

The Zenodo records represent the **archival scientific releases** of the work.

The GitHub repository provides:

- project documentation
- development history
- links to simulation frameworks
- reproducibility resources
