# CQT Simulation Framework

This directory documents the simulation framework used to investigate
multistability and nonlinear dynamics in the CQT mean-field model.

## Zenodo Archive

The complete simulation package is archived at:

https://doi.org/10.5281/zenodo.19100141

The repository contains:

- parameter scan framework
- mean-field solver
- data analysis scripts
- example output data

The GitHub repository documents the simulation architecture and allows
future development of additional numerical experiments.
