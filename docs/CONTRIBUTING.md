# Contributing

Thank you for your interest in the **Coherent Quantum Field Theory (CQT)** repository.

This project is maintained as a structured independent research repository.
At the current stage, contributions are handled conservatively to preserve conceptual consistency and citation integrity.

---

## Scope of Contributions

Constructive contributions may include:

- documentation corrections
- typo fixes
- broken link reports
- formatting improvements
- suggestions for repository clarity
- reproducibility-related feedback

At this stage, major theoretical edits or conceptual rewrites are generally not accepted directly.

---

## Before Contributing

Please first review:

- `README.md`
- `ROADMAP.md`
- `RESEARCH_CONTEXT.md`

This helps ensure that suggestions are aligned with the project structure and intent.

---

## How to Contribute

### Minor changes
You may open an Issue describing:

- the problem
- the exact location
- the proposed improvement

### Larger suggestions
Please open an Issue first before preparing a Pull Request.

This is especially important for:

- structural changes
- simulation-related documentation
- citation-related changes
- metadata adjustments

---

## Pull Request Guidance

If submitting a Pull Request:

- keep the change focused
- explain the purpose clearly
- avoid combining unrelated edits
- preserve scientific tone and internal consistency

---

## Citation and Attribution

If you use ideas, documentation, or software from this repository in academic work, please cite the relevant Zenodo publication.

Main English DOI:
https://doi.org/10.5281/zenodo.19100362

## Licensing

Please note that this repository uses a dual-licensing structure.

- Manuscripts are protected under a non-commercial manuscript license
- Software may be governed by separate license terms
- Commercial use requires explicit authorization where applicable

Review the license files before reusing repository material.

---

## Maintainer Review

All contributions are reviewed manually.
Opening an Issue before larger changes is strongly preferred.
