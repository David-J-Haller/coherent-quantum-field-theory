# Roadmap

## Project Scope

This repository documents the **Coherent Quantum Field Theory (CQT)** research program, including its theoretical framework, manuscript references, and numerical simulation architecture.

The official archival publications are hosted on Zenodo:

- English manuscript: https://doi.org/10.5281/zenodo.19100362
- German manuscript: https://doi.org/10.5281/zenodo.19100136
- Simulation package: https://doi.org/10.5281/zenodo.19100141
